# Hapi.js Template

## Prerequisites

Install **nodemon** module in order to automaticaly restart server when something is changed.

```
npm install -g nodemon
```

## Start app

Clone repository:
```
git clone https://github.com/lukalopusina/hapi-template.git
cd hapi-template
```

Install dependencies:
```
npm install
```


Start server:

```
npm run develop
```

Application url: http://localhost:8000/
